    1. What is CSS

    • CSS stands for Cascade Styling Sheet.
    • CSS describes how HTML elements work on the website.

    2. Selectors & Property

    • CSS Selector selects the element you want to style
    • color, text-align are examples of properties
    • font is also a property

    3. Box Model

    • Box model refers to a box that wraps around every HTML element
    position: static will not be affected by top, bottom, left, right
    position: relative will make the element move relative to the original position
    position: absolute will make the element position itself to the nearest ancestor?
    position: sticky will make the element position stick to a place when given an offset
    position: fixed will make the element position stays in the same place even if the page is scrolled
